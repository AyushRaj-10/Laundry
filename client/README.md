# Client Notes

This `client` folder contains the React + Vite frontend for the laundry management system.

## What Lives Here

- authentication screens for login and registration
- protected dashboard and order history views
- order creation UI
- Context API providers for auth and orders
- shared Axios configuration for authenticated API calls

## Environment

Create `client/.env` with:

```env
VITE_BACKEND_URL=http://localhost:8000
```

## Scripts

```bash
npm run dev
npm run build
npm run preview
npm run lint
```
